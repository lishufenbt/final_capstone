Explanation of parameters

  • Contributing factors

    Contributing factor 1 indicates the police recorded factor by vehicle 1 in a collision that caused the accidents

    Contributing factor 2 indicates the police recorded factor by vehicle 2 in a collision that caused the accidents

  • Vehicle types

    Vehicle type 1 indicates the type of vehicle 1 in a collision

    Vehicle type 2 indicates the type of vehicle 2 in a collision

  • Block code

    Block code indicates the likelihood of accidents in a certain block.

    The values of block code varies between 0-10 but most are below 3, mean is 1.29, 0.5 percentile 1.25, 0.9 percentile 2.23. For comparative likelihood of blocks please refer to the second map in "DATA"

  • Hour, month, street width

    Hour, month and street width are other few quantities used to predict the accidents
